The objective of this laboratory session is to accelerate the computation of the covariance matrix, but using the (Message Passing Interface) \texttt{MPI} library. The \texttt{MPI} library is a standard for parallel programming that allows processes to communicate with each other by sending and receiving messages. It is widely used in high-performance computing applications, where the computational workload can be distributed across multiple processors or nodes.
