The variables $v_1$ to $v_5$ were read from the files \verb|var1.dat| to \verb|var5.dat|. Each variable contains 134217728 elements.

Using these variables, the derived variables $v_6$ to $v_{10}$ were defined as:
\begin{equation}
    \begin{aligned}
        v_6     &= \sin(v_1) + \cos(v_2)\\
        v_7     &= e^{v_3} + e^{-v_4}\\
        v_8     &= \sin(v_1)\cos(v_2) + \cos(v_3)\sin(v_4)\\
        v_9     &= \sqrt{v_2^2 + v_3^2}\\
        v_{10}  &= \sqrt[3]{v_5}
    \end{aligned}
\end{equation}

\begin{itemize}

    \item \verb|-lm|: links the mathematical library required for functions such as \verb|sin|, \verb|cos|, \verb|exp|, and \verb|sqrt|.
\end{itemize}
